The Medicare download data includes prescription claims, but they are only defined by an 11-digit Drug Codeâ€”no mention of the name of the drug.  
  
You can check the accuracy of the prescription by looking it up at [[EXTERNAL LINK] - National Drug Code Directory](https://dps.fda.gov/ndc) (in the Select Type dropdown, choose NDC Code).  
  
For your search term, the Drug Code from the download needs to modified. Replace the 6th digit 0 with a - and, for more precision, add a - between the 9th and 10th digits.  
  
Here's an example using Shingrix to search at FDA:  
58160082311 is the drug code in the download  
58160-823 is the actual NDC Code that you can use for your search  
58160-823-11 is the NDC Code plus packaging info for the prescribed medication  
  
(Alas, not all drugs use the xxxxx-xxx-xx format. For some drugs, you may need to use the format xxxx-xxxx-xx or xxxxx-xxxx-x to return results.)
