The [Progressive Insurance](https://wallethub.com/profile/ci/progressive-insurance-13015814i) ICC code is 24260. The ICC code stands for Insurance Company Code and is designated to an insurance company by the National Association of Insurance Commissioners. The ICC code is commonly referred to as the company's NAIC code. This code is used as a unique identifier by consumers on the NAIC database to:

* View the number of closed complaints for a company
* Report fraud
* Review enforcement actions taken against a company
* Read market conduct examination reports
* Download financial statement data

Consumers can also use a company's three-digit DMV code when they're looking for a company's ICC code. An insurance company's DMV code differs based on each state's DMV, so in order to locate it for Progressive, for instance, you'd have to search for it on your local DMV's website. After you have purchased an insurance policy, you can find the company's DMV code on your insurance card and you may need to use it to register your vehicle.
