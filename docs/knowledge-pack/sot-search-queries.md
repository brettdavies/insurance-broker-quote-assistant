## Search Techniques & Operators

### Google Search Operators

```
site:domain.com           # Search specific site
intitle:"exact phrase"    # Search in page title
filetype:pdf              # Find PDF documents
"exact phrase"            # Exact phrase match
OR                        # Either term
-exclude                  # Exclude term
*                         # Wildcard
2020..2024               # Date range
```

### Advanced Search Patterns

```
# State-specific carrier query
"[Carrier]" "[Product]" "[State]" [DataPoint]
Example: "GEICO" "auto insurance" "California" "average cost"

# Regulatory source query
site:[state-insurance-dept] [topic]
Example: site:insurance.ca.gov minimum coverage

# Discount percentage query
"[Carrier]" "[Discount Name]" percentage OR "save" OR "%"
Example: "GEICO" "multi-policy" percentage OR "save" OR "%"

# Industry benchmark query
site:[comparison-site] [carrier] [state] [metric]
Example: site:bankrate.com GEICO California rates
```

## Usage Guidelines

### For Automated Agents

1. **Start with highest authority sources**: Official carrier sites, state .gov
2. **Use site: operator** to focus searches
3. **Capture element references**: CSS selectors for exact locations
4. **Record all findings**: Even conflicts (resolve later)
5. **Note confidence levels**: High (official), medium (industry), low (aggregator)

### For Manual Research

1. **Follow search query order**: Start with primary, then advanced
2. **Document exact URLs**: Full path, not just domain
3. **Screenshot or save HTML**: For element reference
4. **Note access date**: ISO 8601 timestamp format (YYYY-MM-DDTHH:mm:ssZ) - see [Date and Timestamp Standards](sot-schemas.md#6-date-and-timestamp-standards)
5. **Copy exact wording**: For rawValue field

### Quality Checklist

For each data point found:
- [ ] Full URL recorded
- [ ] Element reference captured (CSS selector or XPath)
- [ ] Access date documented
- [ ] Raw value copied exactly
- [ ] Confidence level assessed
- [ ] Screenshot saved (optional but recommended)
